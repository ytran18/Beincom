## 	Getting Started

<!-- Run Locally -->
###  Run Locally

Clone the project

```bash
  git clone git@github.com:ytran18/Beincom.git
```

Go to the project directory

```bash
  cd Beincom
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
